# **App Name**: SoundSpell Academy

## Core Features:

- Student View Interactive Lessons: An engaging, highly visual interface for students (ages 5-12) to access interactive phonics lessons and activities.
- Parent View Progress Dashboard: A data-focused dashboard for parents to track their child's learning progress, completed lessons, and performance metrics.
- Secure Student & Parent Authentication: User registration and login for both students and parents to ensure personalized access and data privacy.
- Phonetic Content Management: Backend capabilities for administrators or educators to create, edit, and organize phonetic lessons, exercises, and learning materials.
- PostgreSQL Progress Tracking: Store all student progress, lesson completions, scores, and activity data in the PostgreSQL database for reliable reporting.

## Style Guidelines:

- The visual design draws inspiration from concepts of youthful learning and clarity. Primary color: A clear and friendly medium blue (#175CC9). It evokes a sense of trust and intelligence suitable for an educational platform. Its brightness is balanced to contrast effectively with a light background.
- Background color: A very light, desaturated blue (#E7ECF2). This color is subtle and serene, ensuring that foreground content stands out without being stark, and provides a gentle aesthetic for a children's learning environment.
- Accent color: A vibrant cyan (#3BD0E8). This lively hue provides an energetic pop, drawing attention to interactive elements, calls-to-action, and key highlights within the interface, particularly appealing for the student view.
- Headlines and prominent text in 'Poppins' (geometric sans-serif) for its modern, friendly, and precise feel, making it approachable for younger students. Body text and data displays will use 'Inter' (grotesque-style sans-serif), chosen for its excellent readability and neutral appearance, perfect for the detailed parent view and general app content.
- Employ a consistent set of clean, playful, and easily recognizable vector icons for the Student View to guide young users, while opting for more subtle, informational icons for the Parent View dashboard to maintain a professional, data-driven aesthetic.
- Student View features large touch targets, clear visual hierarchy, and ample white space with an intuitive, guided flow. Parent View will utilize a grid-based layout for data presentation, ensuring information is organized and easy to scan, with collapsible sections for detailed views.
- Subtle, encouraging animations will be used in the Student View to provide positive feedback (e.g., correct answer confirmations, transition between lessons). Parent View will have minimal, smooth transitions for data updates and navigation, ensuring a polished and unobtrusive experience.
